Description

We promised robots: it's time for robots! In this stage, you should create your own robots handmade or using websites such as fsymbols.com to help with the design. After finishing your design, hit the "COPY" button and it's ready to be pasted into your code! If you'd like to, you can use an ASCII-image from the example:

  $   $$$$$$$   $
  $$$$$     $$$$$
      $$$$$$$
     $$$   $$$
     $       $

You will also add a menu to the game screen.

Objectives
  1. Your robots are represented by their ASCII/Unicode images, which should be placed between first two lines, started with "+" symbol in hub and be divided by '|' character. In this stage, you should have three robots. Make sure, each line of your robot image contains a non-whitespace character
  2. Place the robots into the Hub you made earlier. Make sure each robot's string representation is rectangular.
  3. Underneath the robots, insert 4 options just like in the examples below:
    • [Ex]plore
    • [Save]
    • [Up]grade
    • [M]enu
    where the text in brackets represents what the player needs to type in to execute that command. We will be implementing these options in later stages.
  4. For now, just add the [M]enu to our game screen. The menu should have the following options:
    • [Back] to game
    • Return to [Main] Menu
    • [Save] and exit
    • [Exit] game
    Remember, for not-yet implemented options display a placeholder message (for example, Coming SOON!) and terminate the program.
  5. The menu options should do what they say, except for [Save] and [Exit]: just insert a placeholder message there and end the program. We will deal with saving later.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
